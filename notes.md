React JS Learnings
1. Create React App (CRA)

Creates a React project with all configuration already set up.

2. React Router

Used for navigation between pages without reloading the browser.

3. BrowserRouter

Enables routing in a React application.

4. Routes

Container that holds all route definitions.

5. Route

Connects a URL path to a React component.

6. Component

A reusable piece of UI such as Navbar, HeroSection, or Footer.

7. HomeScreen

A page wrapper that combines multiple components.

8. React Fragment (<> </>)

Groups elements without adding an extra <div> to the DOM.

9. Functional Component

A JavaScript function that returns JSX.

10. JSX

HTML-like syntax written inside JavaScript.

11. className

React uses className instead of class.

12. Public Folder

Stores static assets like images, icons, and PDFs.

13. CSS Variables (:root)

Reusable values for colors and theme management.

14. Utility Classes

Reusable styles for buttons, text, and spacing.

15. CSS Grid

Used to create responsive two-column layouts.

16. 1fr

One fraction of available space; repeat(2, 1fr) creates two equal columns.

17. Hover (:hover)

Changes styles when the mouse is over an element.

18. Transition

Makes CSS changes smooth and animated.

19. Responsive Image

width: 100% makes the image scale with its container.

20. PascalCase

React component names should start with a capital letter (HeroSection, HomeScreen).
